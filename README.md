Schoenstatt 3.0max app

A minimal mobile-first web app for Schoenstatt Nigeria community updates, events, and media.

🚀 Features
- View posts, events, and photos
- RSVP to events
- Like and comment on photos
- Profile and settings section

🛠 Built With
- HTML
- CSS
- JavaScript (vanilla)
- firebase(backend)
- No framework

📁 Project Structure

├── index.html
├── /styles
├── /scripts
├── /files
└── /public
